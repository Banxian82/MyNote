\subsubsection{类型转换}
类型转换是将一个数据类型的值转换为另外一个数据类型。

\begin{itemize}
	\item 隐式类型转换: 自动类型转换，通常是将小的数据类型转换为大的数据类型。
	\item 显式类型转换: 该类型转换需要使用强制类型转换符，将一种类型的值强制转换为另一种类型。
\end{itemize}
